Proyecto de e-commerce creado durante el programa JAP. Partiendo de una estructura base, nos encargamos del desarrollo de funcionalidades clave del frontend y backend, integración de datos, diseño de la base de datos y consumo de la API necesaria para el funcionamiento del carrito y la visualización de productos.

**Integración de datos del catálogo:** obtención y renderizado dinámico de los productos, organizados por categorías.

**Detalle de producto:** desarrollo de la página individual con información completa del producto.

**Personalización del frontend:** modificaciones al navbar y ajustes en la interfaz general para mejorar la navegación.

**Página de perfil de usuario:** implementación de la vista de perfil con edición de datos básicos.

**Creación de la base de datos:** diseño de tablas principales y relaciones necesarias para productos, usuarios y carrito.

**Consumo de la API propia:** implementación de llamadas GET para obtener datos desde la base y un POST para manejar el carrito de compras.
